# Produtos e Exames - Lucid Trading

A Lucid possui três categorias principais de investimento/exame. Se o usuário perguntar "quais tipos", liste estes três:

## 1. LucidPro (Evaluation)
É o modelo de avaliação padrão para traders.
*   **Tamanhos de Conta:** $25k, $50k, $100k, $150k.
*   **Drawdown:** End-of-Day (EOD).
*   **Meta de Lucro:** Ex: $3,000 para conta de $50k.
*   **Limites:** Dias mínimos de negociação (5 dias).

## 2. LucidFlex
Modelo com regras de drawdown mais flexíveis ou específicas para diferentes perfis.
*   **Destaque:** Utiliza sistema de End-of-Day Drawdown.
*   **Perfil:** Traders que precisam de mais "folga" no saldo intraday.

## 3. LucidDirect (Financiamento Direto)
Modelo onde o trader pula a fase de avaliação e vai direto para a conta financiada.
*   **Destaque:** Sem teste/desafio.
*   **Condição:** Taxas de ativação maiores ou regras de participação de lucro diferentes.
